机械外贸官网 Next.js 开发需求文档（AI IDE 直接可用版）
## 1. 项目整体说明
项目名称：机械产品外贸B2B官方展示官网
项目定位：面向欧美市场（主打美国）割草机/园林机械工厂官网，用于品牌展示、产品陈列、客户在线询价、在线客服对接、获取海外询盘，纯企业展示型站点，无商城交易功能。
开发身份：自有程序员开发，拒绝 WordPress，全程手写自研前端，轻量化、高性能、无冗余代码。
核心目标：Google SEO友好、海外访问极速、移动端自适应、自带询盘报价能力、可长期迭代、全自动部署上线。
## 2. 强制技术栈（固定不可改）
- 框架：Next.js 当前项目已经初始化，无需重新初始化
- 渲染模式：纯静态 SSG 预渲染（禁止 SSR / 禁止 CSR 纯SPA）
- 样式方案：Tailwind CSS
- 语言：TypeScript 全量TS、严格类型
- 数据管理：产品/分类/公司信息全部使用本地 JSON 文件管理，不搭后台、不连数据库
- 图标：Lucide / Heroicons 轻量化图标
- 国际化：Next.js 原生 i18n 中英双语（默认英文，可切换中文）
- 打包模式：静态导出，无服务端依赖
## 3. Next.js 固定工程配置（AI 直接照着写）
3.1 next.config.js 强制配置
必须开启纯静态导出，解决路由404、图片兼容问题：
/** @type {import('next').NextConfig} */
const nextConfig = {
  output: 'export',
  trailingSlash: true,
  images: {
    unoptimized: true
  },
  reactStrictMode: true
}

module.exports = nextConfig
3.2 打包命令
package.json scripts：
"scripts": {
  "dev": "next dev",
  "build": "next build",
  "start": "next start"
}
打包产物输出目录：根目录/out
3.3 路由404解决方案
项目 public 目录新建 _redirects 文件（无后缀）
/*    /index.html   200
## 4. 部署架构（永久免费、无服务器）
部署方式：Cloudflare Pages 全自动部署
托管模式：纯静态托管，零服务器、零主机、零运维
CI/CD：推送 GitHub 自动打包、自动部署、自动更新官网
CDN：Cloudflare 全球免费CDN、自动SSL、强制HTTPS、Brotli压缩、图片WebP优化
域名：自有国际 .com 域名，NS托管在Cloudflare
## 5. 网站页面结构（全部需要开发）
全站适配 PC + 移动端自适应，工业简约高端外贸风格，白色/深灰主色调，干净专业。
5.1 首页 Home
- 全屏Banner 工业风轮播
- 核心产品分类快捷入口
- 工厂优势、核心技术、产品特点简介
- 热销产品展示模块
- 工程案例/应用场景展示
- 合作客户、资质简介
- 底部联系、询盘入口
5.2 产品中心 Products（核心模块）
- 产品多级分类筛选
- 产品列表页：缩略图、名称、简介、型号
- 产品详情页：大图展示、参数表格、特性、应用场景、视频占位、PDF说明书下载入口
- 每个产品详情页内置 立即询价按钮
产品数据全部读取本地 JSON，后期新增产品只需改JSON，无需改代码。
5.3 案例展示 Cases
机械施工场景、海外项目案例、图片画廊展示
5.4 关于我们 About
工厂介绍、生产实力、资质证书、企业优势、工厂实景图
5.5 联系我们 Contact
- 公司地址、邮箱、WhatsApp、电话

## 6. 核心功能需求（必做）
6.1 中英双语国际化
- 默认访问英文界面（适配海外用户）
- 顶部切换中英文
- 所有页面、按钮、表单、弹窗双语全覆盖
- 语言文件本地json管理
6.2 在线客服功能（第三方嵌入）
- 全站右下角悬浮在线客服按钮
- 预留 JS 嵌入位，后期接入 Tidio / Crisp 外贸客服（无需自研、无需服务器承载）
- 支持离线留言、在线聊天、WhatsApp跳转
6.3 SEO 全站优化（外贸刚需）
- 每页面独立 Title、Meta Description、Keywords
- 自动生成 sitemap.xml、robots.txt
- 图片 alt 属性齐全
- 静态化页面利于谷歌收录
- 规范路由、无冗余参数
6.4 性能优化
- 图片懒加载
- 静态资源压缩
- 无冗余依赖、无多余插件
- 全站轻量化，海外极速打开
##7. 数据管理规则（极简运维）
- 所有产品数据、分类、文案、公司信息 全部 JSON 本地化
- 不使用任何后台管理系统、不使用CMS、不依赖数据库
- 更新内容流程：改JSON → git push → 自动部署更新网站

4. 全站SEO配置、sitemap、robots
5. 自适应移动端+PC完整页面
